# VSA 2 - Vrije studieactiviteiten
## Opdracht omschrijving
Vrije studieactiviteiten zijn activiteiten, waarbij een zelf gekozen of geworven opdracht wordt uitgewerkt. Deze opdracht maakt geen onderdeel uit van het standaard curriculum, maar moet wel binnen het ICT-werkveld vallen. Een VSA heeft een standaardomvang van 28 klokuren.
